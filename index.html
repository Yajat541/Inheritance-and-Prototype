Q1. Create 2 objects - parent and child. Attach methods to parent and use those methods in child object using parents prototype 
        const parent={
        display: function(){
        console.log(`${this.name} and age is ${this.age}`)
        }
        }
        const child=Object.create(parent)
        const child.name="Yajat Dhawan"
        child.age=22
        child.display();

>OUTPUT :-Yajat Dhawan and age is 22

Parent object passed display function. Child Object inherit the property of parent and passed two keys and values name, age , after that child Acessed the parent display function form child

            }

Gfather property inherit to father and father property to You . You can acess the property of father and Gfather also.
